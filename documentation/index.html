About LARAVEL C.R.U.D. with Modals & AJAX

LARAVEL C.R.U.D. with Modals & AJAX help developers who love to work with Laravel and Bootstrap to create a simple CRUD operations  which by default will require user to navigate from one page to another for simple operations.

Modals are streamlined, but flexible, dialog prompts with the minimum required functionality and smart defaults, and therefore seem an excellent fit for all forms of forms and are very useful for implementing CRUD operations and on the other hand, by using modal dialogs users can use the same action (methods) and keep them on same page.

Setting up your MySQL

This script requires PHP 5.4+ and MySQL. 5.x

Now, first thing you want to do, is to make sure you create a MySQL database along with an user for it (IMPORTANT NOTE: Copy somewhere your mysql username, and password, as you will need them later), and setup your FTP server for the domain where the script will be installed.

So your requirements are:

  • A) A MySQL database + MySQL user
  • B) Domain name + FTP server.

Upload process

  • The first step is to upload the content inside the folder "laravel-crud" (IMPORTANT NOTE: Upload what's inside the folder Script, not the script folder) into your public_html folder or into your default FTP root folder.
  • The second thing is to upload into your MySQL database using the IMPORT function available in MySQL the ajax.sql file from the folder called "data/sql".

Installation process

With a text editor (e.g.: Notepad++), open the following file located inside the 'script' folder:

PATH: script-name/app/config/database.php

Now, replace what's inside the array with your information, you should have something like this:

        'mysql' => array(
            'driver'    => 'mysql',
            'host'      => '',
            'database'  => '',
            'username'  => '',
            'password'  => '',
            'charset'   => 'utf8',
            'collation' => 'utf8_unicode_ci',
            'prefix'    => '',
        )

How to use

First steps was for testing this app to see how it works, right now I am going to explain how to use/insert code in your application.
Before your start, don't forget to create a back-up of your files.

The project will be set up as follows:
- modify your model as you see in models/Client.php
- modify your controller as you see in controllers/ClientController.php
- insert in your views the folder named _assets, or copy anywhere you want folder modals located in _assets (keep in mind the location of this one)

Now we need to implement modals and features related with these
- go to your views/your_folder_where_you_need_to_implement_MODAL, I suppose you have the next files and use Blade Templating
    - index.blade.php
    - create.blade.php
    - edit.blade.php
    - show.blade.php

STEP 1.
- remove from create, edit, show
    - @extends('.....')
    - @section('content')
    - @stop

STEP 2.
- include modals in index.php
- remove your "table code" from index.php and create a new file called table.blade.php with your "table code"
- adapt your buttons for modals and events    

STEP 3.
- copy the javascript code from /* === MODALS === */ ...... /* === END MODALS === */

Don't worry, extra information you will find in each file, I created an helpful comments.
